Criminal Court Proceedings and Trials

What's this?

This is the site of a project that aims to improve the way MOJ publishes open data. The plan is:

  • publish a single dataset in raw form
  • work with people with an interest in the dataset to understand their needs
  • from those identified needs, improve the way we serve the data
  • iterate until the service meets expectations

The success of this project will hopefully lead to a process for MOJ to publish data better, and incentive for us to offer more datasets in the same way.

The dataset

This site lets you access the list of criminal court outcomes from 2002 to 2012 (only adult defendants). A detailed description of the data is available.
